Juego POO - Pokémon Battle

Proyecto de Programación Orientada a Objetos (POO) desarrollado en Python.


 Descripción

Este es un juego de batallas por turnos inspirado en Pokémon, implementado utilizando los conceptos principales de Programación Orientada a Objetos:
- Herencia
- Polimorfismo
- Encapsulamiento
- Abstracción

 Características

- Sistema de batallas por turnos
- Múltiples Pokémon con diferentes estadísticas
- Sistema de experiencia y niveles
- Tabla de mejores jugadores (ranking)
- Interfaz por consola interactiva

 Cómo ejecutar el juego

Requisitos
- Python 3.12.0

Pasos

1. Clona o descarga el repositorio.
2. Abre una terminal en la carpeta del proyecto.
3. Ejecuta el siguiente comando:

comando para ejecutar el juego
-python juego.py
